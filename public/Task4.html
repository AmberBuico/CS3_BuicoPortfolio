Reflection

For me, the easiest event was the click event because it's the most straightforward...it only needs a single function to run exactly once when the button is pressed.

The trickiest was the keypress event. While it works easily, the challenge was remembering that I needed to pass the `event` object to the function and then access `event.key` to actually see which button I typed. It required an extra step to get the information I wanted.

Overall, the mouseover/mouseout pair felt the most powerful because it let me create a two-part effect (rotate and un-rotate), making the element feel truly interactive :DDDD
