Reflection

The difference between mouseover and mouseout is that mouseover is the event that runs the code when the mouse moves onto the element.

Mouseout is the opposite! It runs the code when the mouse moves off the element.

We need both of these events for the "Hover to Zoom" to work correctly. Mouseover makes the box change and zoom in, and mouseout brings it back to its normal, starting state.
